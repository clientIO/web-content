We've published a new demo: [Electric Generator](https://www.jointjs.com/demos/electric-generator)

Do you want to animate elements in a diagram? You can use the [transition](https://resources.jointjs.com/docs/jointjs/v3.6/joint.html#dia.Element.prototype.transition) feature on the model attributes or write a custom element view and take advantage of the fact that JointJS shapes are made up of SVG, which can be set in motion using the [Web Animation API](https://developer.mozilla.org/en-US/docs/Web/API/Web_Animations_API). See the latter approach in action in this [CodePen](https://codepen.io/jointjs/pen/poKRaRY/321e97edc7793da01a46e701324f1557).
