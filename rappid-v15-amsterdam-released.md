[JointJS+](/jointjs-plus), the **HTML 5 diagramming UI framework**, version v1.5 just landed. The new version does not only improve the current components but brings **ten brand new plugins** to make your apps even better.

We'd like to thank to our clients for their extremelly valuable feedback!

## New features

### Inline Text Editor

JointJS+ finally contains an [inline text editor](https://resources.jointjs.com/docs/rappid/v3.5/ui.TextEditor.html). The editor can be applied to any text in both elements and link labels.

The text editor even works on a text on a curved path.

### Tree Layout

[Automatic Tree Layout](https://resources.jointjs.com/docs/rappid/v3.3/layout.html#layout.TreeLayout) is great for displaying and manipulating trees. Two plugins are available related to tree layouts. One is purely for auto-layoting graphs into trees and the other one, [ui.TreeLayoutview](https://resources.jointjs.com/docs/rappid/v3.3/layout.html#layout.TreeLayout) implements common interaction with trees such as attaching and dettaching a branch.

A demo is available [here](https://www.jointjs.com/demos/organizational-chart).

### Shortest Path Finding

[alg.Dijkstra](https://resources.jointjs.com/docs/rappid/v3.5/alg.Dijkstra.html) plugin together with a [alg.PriorityQueue](https://resources.jointjs.com/docs/rappid/v3.5/alg.PriorityQueue.html) and the graph extension via [dia.GraphUtils](https://resources.jointjs.com/docs/rappid/v2.3/dia.html#dia.GraphUtils) give you an easy and quick way of finding the shortest paths in your graphs.

A demo is available [here](https://www.jointjs.com/demos/shortest-path-algorithm).

### Modal and Non-Modal Dialog Boxes

[ui.Dialog](https://resources.jointjs.com/docs/rappid/v3.5/ui.Dialog.html) plugin brings both modal and non-modal dialog boxes. A default set of common boxes is included (alert, info, success, warning and neutral), an arbitrary HTML can be used to display any content, even diagrams!

### Matrix Charts

[Matrix chart](https://resources.jointjs.com/docs/rappid/v3.5/shapes.chart.html#matrix) is great for visualization of co-occurences in a matrix.

### Navigator View

[Navigator](https://resources.jointjs.com/docs/rappid/v3.5/ui.Navigator.html) is a small widget displaying large diagram in a small format with a pannable and resizable rectangle.

### Local Storage

[Local Storage](https://resources.jointjs.com/docs/rappid/v3.1/storage.html) plugin gives you a high-level API for storing graphs, element, links and any other documents in the HTML localStorage.

## ... and much more!

The [JointJS core API](https://resources.jointjs.com/docs/jointjs/v3.6/joint.html) got a great bunch of new methods. Vectorizer now supports [texts along paths](https://resources.jointjs.com/docs/jointjs/v3.6/joint.html#v:text) and more. See the JointJS+ v1.5 Amsterdam CHANGELOG page for details.
