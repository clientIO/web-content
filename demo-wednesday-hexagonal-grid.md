We've published a new demo: [Hexagonal Grid](https://www.jointjs.com/demos/hexagonal-grid)

How would you create a hexagonal grid diagram? We used the [Honeycomb library](https://github.com/flauwekeul/honeycomb) to do the math for us, and JointJS, which rendered the grid and the shapes in it and made everything interactive.

If you want to learn more about hexagonal grids, also check out this article - <https://www.redblobgames.com/grids/hexagons/>.
