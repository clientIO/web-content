We've published a new demo: [Searchable Sitemap](https://www.jointjs.com/demos/searchable-sitemap)

Today's demo will help you understand how to generate a searchable interactive sitemap from an XML file - using a custom search instead of a native browser search.

Although JointJS+ diagrams (thanks to the SVG approach) allow you to use browser search to find text in elements and scroll to them, there are cases where you need to implement custom search. Imagine searching for content that is hidden in a collapsed branch or when virtual rendering is in place. A custom search would be the only way to go, which is nicely demonstrated in the example below.

See the Pen [JointJS+: Searchable Sitemap](https://codepen.io/jointjs/pen/VwEOaNr) by JointJS ([@jointjs](https://codepen.io/jointjs))
