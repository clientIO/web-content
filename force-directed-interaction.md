# JointJS: Force Directed Interaction

This demo showcases a force-directed layout with interactive capabilities. While the simulation is running, users can interact with elements—adding new ones or modifying existing positions. The layout dynamically adapts to these changes, updating in real time to reflect user input.

This demo is also available online at [jointjs.com](https://jointjs.com/demos/force-directed-interaction).
