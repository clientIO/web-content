JointJS+: Dynamic Stencil <a href="https://www.jointjs.com/jointjs-plus"><img src="../jointjs-plus-badge.svg" alt="JointJS+" width="123" align="right" /></a>

The Dynamic Stencil demo provides the user with a palette of JointJS elements which can be dragged onto the paper. The user is able to manipulate selections, and dynamically add the selections to the palette for future use.

This demo is also available online at jointjs.com.
