# Visio BPMN Import

The Visio BPMN Import demo imports a Microsoft Visio VSDX file and converts Visio Shapes to JointJS cells.

This demo is also available online at [jointjs.com](https://jointjs.com/demos/visio-bpmn-import).

## Available Versions

- [JavaScript](./js/)
- [TypeScript](./ts/)
